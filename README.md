# Zaara Perfumes - Wholesale Fragrance Oils

A premium wholesale e-commerce platform for Zaara Perfumes, built with Next.js, Tailwind CSS, and Lucide Icons.

## ‚ú® Features

- **Wholesale Multi-Tier Pricing**: Economy, Inspired, and Identical (Identical) grades.
- **Large Catalog Support**: Efficiently handles 1600+ fragrance oils.
- **List View Ordering**: Toggleable list view for rapid bulk selection.
- **Cart Inquiry System**: Generates a detailed WhatsApp inquiry with product lists and price estimates.
- **Advanced Search**: Real-time filtering by collection and size.
- **Interactive Branding**: Premium "soft-square" logo integration with subtle animations.

## üöÄ Deployment (Vercel)

This project is optimized for deployment on [Vercel](https://vercel.com).

### 1. Push to GitHub
If you haven't already:
1. Create a new repository on [GitHub](https://github.com/new).
2. Open your terminal in this project folder.
3. Run the following commands:
   ```bash
   git init
   git add .
   git commit -m "Initial commit from Antigravity"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
   git push -u origin main
   ```

### 2. Connect to Vercel
1. Go to [Vercel](https://vercel.com/new).
2. Import your GitHub repository.
3. Click **Deploy**.
